# NodeJS-Express-Memory-Grapher
NodeJS Express Memory Grapher

Setup:
Create JSON file `public/js/data.json` formatted as an array of hashes:
```
[
  {"datetime":memory},
  {"datetime":memory}  
]
```

Build:
```
npm install && bower install
```

Run:
```
node app.js
```

Browse:
```
http://localhost:3000
```
